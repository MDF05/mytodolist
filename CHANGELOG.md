# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [Unreleased]

### Added
-   Initial project structure.
-   Core navigation with `expo-router`.
-   Basic note-taking functionality.
-   Local storage integration with `async-storage`.

## [1.0.0] - 2026-01-14

### Added
-   Launched first stable version of **MDF Note**.
-   Support for creating, editing, and deleting notes.
-   Dark/Light mode support based on system settings.
-   Search functionality for notes.
-   Comprehensive documentation system.

### Changed
-   Updated `react-native` to v0.76.7.
-   Updated `expo` to v52.0.37.
